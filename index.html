FAQ

Questions You Probably Have

How customizable are the pages really?

You control everything via your prompt. Colors, fonts, layout structure, copy tone, sections to include. Want a specific headline? Include it. Want your brand colors? Tell the bot the hex codes. The AI builds exactly what you describe.

What if I need to edit the code manually?

Every page is saved to your GitHub repository. You have full access to the HTML, CSS, and JavaScript. Edit it directly, hand it to a developer, or keep using the bot for updates. You own the code completely.

Do I need technical skills to set this up?

No. If you can add a bot to Slack or Discord, you can use InstantPage AI. The setup takes about 2 minutes. No API keys to configure. No servers to manage. No DNS to understand (though you'll need to add a DNS record for custom domains, which we walk you through).

Can I use this for client work?

Absolutely. That's the main use case. Build pages for clients, host them on their domains, hand over the GitHub repo when the project ends. Full white-label. Your clients never need to know InstantPage AI exists.

What about page speed and SEO?

Pages are static HTML hosted on Cloudflare's global CDN. They load fast everywhere. Clean code structure. Mobile-responsive by default. Meta tags and Open Graph included. Everything you need for search and social.

What happens to my pages if I cancel?

They're in your GitHub repo. They're yours forever. If you want to keep hosting them, you can deploy them anywhere (Netlify, Vercel, your own server) since you own the code. We don't hold your pages hostage.
